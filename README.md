
# Iconos

## Usage/Examples

```javascript
import icons from "@Lukkas98/icons"
import { javascript } from "@lukkas98/icons/technologies"

function App() {
  return (
  <div>
    <img src={icons.javascript} />
    <img src={javascript} />
  </div>
  )
}
```
